# Dwarf Fortress Floor Portrait Studio v1.0.0

The first public release turns pixel-art ideas and imported images into practical Dwarf Fortress constructed-floor plans of up to 100×100 tiles.

## Highlights

- 115-material planning catalog with original procedural textures.
- Responsive canvas editor with six drawing tools and undo/redo.
- Color or texture previews plus image quantization and dithering.
- Validated provider-neutral copy/paste workflow for AI-chatbot-generated compact designs.
- PNG, JSON, ordered CSV, compact build runs, text plans, and staged Quickfort CSV downloads.
- Local-only persistence: no account, API key, or cloud service.

## Quickfort limitation—read this first

Quickfort `Cf` cells encode floor placement, not floor material. The staged CSV therefore starts with a harmless README and then provides one section per material. Set buildingplan’s exact floor filter before applying every section, and keep the same upper-left anchor. Metal phases also require `buildingplan set bars true`.

See `docs/QUICKFORT.md` for the complete workflow.

## Download

Download `dwarf-fortress-floor-portrait-studio-v1.0.0.zip`, verify it against `SHA256SUMS`, extract it, and serve the extracted directory:

```bash
python3 -m http.server 8000
```

Then open <http://127.0.0.1:8000/>.
