Dwarf Fortress Floor Portrait Studio is an independent, unofficial fan utility. It is not affiliated with, endorsed by, or sponsored by Bay 12 Games, Kitfox Games, or the DFHack project.

## Visual assets

The application contains **original procedural texture artwork** generated by `scripts/generate-materials.mjs`. The public repository and release artifacts do not contain the third-party reference sheet used during the private prototype or crops from that sheet.

The project’s original procedural textures are released under the repository’s MIT License.

`docs/images/dwarf-fortress-floor-portraits.png` is an in-game Dwarf Fortress screenshot provided by the repository owner to illustrate the intended constructed-floor result. It contains Dwarf Fortress game visuals and user-interface elements. The screenshot is not original project artwork, is not covered by the repository's MIT License, and is included only for descriptive, non-affiliated fan-project context. Rights in the depicted game content remain with their respective owners.

## DFHack and Quickfort

The project emits CSV text intended for DFHack Quickfort. DFHack is a separate project and is not bundled here. Quickfort material limitations are documented in `docs/QUICKFORT.md` and are based on the public DFHack documentation.

## npm dependencies

Runtime and development dependencies are declared in `package.json` and locked in `package-lock.json`. They remain under their own licenses. Notable runtime projects include React, Lucide, Alegreya SC, and IBM Plex fonts distributed through Fontsource packages.

`THIRD_PARTY_LICENSES.txt` contains the copyright notices and complete license texts for every npm runtime dependency included in the production bundle, including the SIL Open Font License notices required for the bundled font files. Regenerate it with `npm run licenses:generate` and verify it with `npm run licenses:check`.
